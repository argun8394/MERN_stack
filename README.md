# MERN Stack Project
  
* React.js
* Node.js(Express.js)
* React-router
* Axios
* JWT
* Permissions 
* MongoDB
* Mongoose
* Tailwind
* Vite

## Install the dependencies
 ```bash
yarn
# or
npm install
```
  

In the project directory, you can run:

### for Frontend `npm run dev`

### for Backend `node index.js` or  `nodemon`
